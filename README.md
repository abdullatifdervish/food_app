## Türkçe

### 📱 Proje Hakkında
Bu proje, kullanıcıların popüler yemekleri listeleyebileceği, sepetlerine ürün ekleyip güncelleyebileceği, modern ve şık bir arayüze sahip bir **Yemek Sipariş Uygulamasıdır**. Flutter'ın güçlü widget yapısı kullanılarak temiz, performanslı ve kullanıcı dostu bir deneyim sunmak amacıyla geliştirilmiştir.

### ✨ Öne Çıkan Özellikler
* **Dinamik Popüler Kartlar:** Özel gradyan geçişleri ve şık kart tasarımları.
* **Gelişmiş Sepet Mantığı:** Ürünlerin sepette olup olmadığını kontrol eden, mükerrer eklemeyi önleyen akıllı sepet sistemi.
* **Kullanıcı Bildirimleri:** Sepete ürün eklendiğinde veya zaten var olduğunda kullanıcıyı uyaran SnackBar entegrasyonu.
* **Responsive Tasarım:** Farklı ekran boyutlarına uyumlu arayüz.

### 🛠️ Kullanılan Teknolojiler
* **Framework:** Flutter (Dart)
* **State Management:** SetState / Global State yönetimi

## English

### 📱 About The Project
This project is a modern and stylish **Food Delivery & Ordering Application** where users can browse popular dishes, add products to their cart, and manage their orders. It was developed using Flutter's powerful widget tree to provide a clean, performant, and user-friendly experience.

### ✨ Key Features
* **Dynamic Popular Cards:** Custom gradient backgrounds and sleek food card designs.
* **Advanced Cart Logic:** Smart cart system that checks if an item is already in the cart to prevent duplicate entries.
* **User Notifications:** SnackBar integration to notify users when an item is added or already exists in the cart.
* **Responsive Layout:** UI that perfectly adapts to different screen sizes.

### 🛠️ Technologies Used
* **Framework:** Flutter (Dart)
* **State Management:** SetState / Global State management
